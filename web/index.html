  1. Extract the installer archive anywhere you like (you may delete it after the installation)

Linux

  1. Extract the installer archive and copy packsly folder to the MultiMC directory so it's on the same level as bin
  2. Open terminal in the packsly folder and execute command mono Packsly3.Cli.exe install
